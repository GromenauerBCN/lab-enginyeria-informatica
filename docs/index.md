# Laboratori de Sistemes — Manel Carrere

Aquest repositori conté tota la documentació, scripts, exercicis i configuració del meu laboratori de sistemes i xarxes creat amb VMware Workstation.

## Objectiu
Disposar d’un entorn complet per practicar:
- Windows Server (Active Directory, DNS, DHCP, GPO)
- Debian 13 (administració Linux, serveis, automatització)
- Xarxes virtualitzades
- Seguretat
- Automatització (bash, PowerShell, Python)
- Cloud (AWS, Azure)
- Exercicis alineats amb el grau d’Enginyeria Informàtica
